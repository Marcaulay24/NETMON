# NetMon Pro v2.1 Improvements & Roadmap

## 1. Feature: Persistent Device Storage (SQLite)
Currently, discovered devices are lost when the application is closed. Implementing an SQLite backend will allow for historical tracking of network changes.

```python
import sqlite3

def init_db():
    conn = sqlite3.connect('netmon_history.db')
    c = conn.cursor()
    c.execute('''CREATE TABLE IF NOT EXISTS devices
                 (ip TEXT PRIMARY KEY, mac TEXT, hostname TEXT, vendor TEXT, first_seen TEXT, last_seen TEXT)''')
    conn.commit()
    conn.close()
```

## 2. Feature: OS Fingerprinting (Nmap Integration)
Enhance the "Deep Scan" functionality by detecting the Operating System of the target device.

```python
def detect_os(self, ip):
    if NMAP_AVAILABLE:
        nm = nmap.PortScanner()
        nm.scan(ip, arguments='-O')
        if 'osmatch' in nm[ip]:
            return nm[ip]['osmatch'][0]['name']
    return "Unknown"
```
